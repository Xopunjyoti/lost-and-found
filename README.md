# 🔍 Lost & Found Hub

A community-driven platform designed to reunite people with their lost belongings. Whether you've lost a set of keys or found a stray pet, this platform simplifies the reporting and matching process.

---

## ✨ Features

* **Post a Listing:** Easily create "Lost" or "Found" posts with descriptions and categories.
* **Image Uploads:** Attach photos to help identify items quickly.
* **Privacy First:** Users can choose what contact information is visible to the public.

---

## 🛠️ Tech Stack

| Component      | Technology |
| :---           | :---       |
| **Frontend** | React.js / Tailwind CSS |
| **Backend** | Node.js / Express |
| **Database** | MongoDB |
| **Auth** | Firebase Auth / JWT |
